# CodeAlpha Internship - Task 2:Exploratory Data Analysis

## Project Overview
Performed an in-depth Exploratory Data Analysis on the Top 40
Highest-Grossing Movies of All Time dataset scraped from Wikipedia.

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
- 40 Top Grossing Movies (1999–2025)
- Features: Rank, Title, Year, Worldwide Gross

## Key Findings
- Avatar (2009) is the highest grossing movie at $2,923.71M
- 2019 was the most dominant year with 7 blockbusters
- Average gross earnings of top 40 movies is $1,437.56M
- Box office earnings show an upward trend over the decades

## Visualizations
- Top 10 Highest Grossing Movies (Bar Chart)
- Number of Blockbusters Per Year (Bar Chart)
- Average Gross Trend Over Years (Line Chart)
- Distribution of Gross Earnings (Histogram)
- Year vs Gross Earnings (Scatter Plot)
